# MyFirstAngular-AirLine Management System

Airline Management application is used to manage information related to various Airline Providers across GEOâ€™s. This project implements the different functions, which are related to Airline Management System. This project is mainly for demonstrating Create, Read, Update and Delete operations (CRUD). Airport authorities can view the list of existing Airlines, add a new Airline, and modify the details of a specific Airline (Eg: Changing providerCode, providerType or any other information related to an Airline). Finally, it is also possible to remove an existing Airline.

## Live Demo

https://myfirstangular-airline.web.app

## Features

#### *User can `create` a Flight
#### *User can `update` an existing Flight
#### *User can `delete` a flight
#### *User can `view` the number of flights created
#### *User can `filter` the flights based on Provider Type(Domestic or International)
